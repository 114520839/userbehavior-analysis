# 淘宝用户行为分析

## 项目背景
基于阿里巴巴天池 UserBehavior 数据集，分析淘宝用户在 2017年11月25日～12月3日期间的行为规律，为运营策略提供数据支持。

## 数据来源
阿里巴巴天池 - UserBehavior Dataset（读取前100万条记录）  
字段：user_id · item_id · category · behavior（pv/fav/cart/buy）· timestamp

## 分析模块
1. **漏斗分析**：浏览→加购→收藏→购买，整体转化率 2.29%，加购→购买转化率 37.03%
2. **用户活跃度**：小时级 + 日级 PV/UV 趋势分析，识别活跃高峰时段（13~14点）
3. **RFM 用户分层**：基于 Recency + Frequency 双维度分层，识别重要价值用户（43.1%）
4. **SQL 验证**：将清洗后数据写入 SQLite，用 SQL 独立验证漏斗转化率与 RFM 计算结果

## 主要结论
- 购买漏斗流失主要发生在浏览→加购环节，加购用户购买意愿强，是促转化的核心人群
- 用户活跃高峰在 13~14 点，建议将推送与促销集中在该时段
- 重要价值用户占比 43.1%，流失用户占 21.5%，具备召回空间
